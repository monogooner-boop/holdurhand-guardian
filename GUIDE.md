I automatically run every time you start your computer. I'll check your projects folder and make sure everything is tucked into GitHub safely.

## 2. Inviting me to a new project 🧶
If you start a new project (a new folder), just make sure it has "Git" set up. If you don't know how, just ask Jonas or SuperGeminiJOnas: "Hey, can you help me invite Bob to this new folder?"

## 4. Feeling stuck? 🫂
